## ğŸ› ï¸ Troubleshooting

### Problem: "No browser window found!"

**Cause**: Browser not detected

**Solutions**:
1. Make sure browser is open and visible
2. Supported browsers: Chrome, Firefox, Edge, Safari, Opera, Brave
3. Window title must contain browser name
4. Try clicking on browser before capturing

---

### Problem: AI captures wrong window

**Cause**: Multiple browser windows open

**Solutions**:
1. Close extra browser windows
2. Make sure the quiz/assignment is in the active tab
3. Click on the browser window before capturing

---

### Problem: Text not extracted correctly

**Cause**: Page formatting or image-based text

**Solutions**:
1. Make sure text is actual text (not images)
2. Zoom in for better clarity (Ctrl + Plus)
3. Try capturing again with better lighting/contrast
4. Some PDFs or images may not read perfectly

---

### Problem: AI gives wrong answers

**Cause**: AI misunderstood or needs more context

**Solutions**:
1. Use "Ask Question" to clarify specific points
2. Capture again with better view of the question
3. Provide more context in your question
4. Remember: AI is a learning assistant, verify important answers

---

### Problem: Too slow to process

**Cause**: Large models or complex page

**Solutions**:
1. Wait patiently - quality takes time (1-2 minutes)
2. Make sure no other heavy programs are running
3. First question is always slower (model warmup)
4. GPU makes it much faster (NVIDIA cards)
